1. Pengertian OOP

OOP adalah paradigma pemrograman yang berfokus pada objek. Objek memiliki data (state) dan perilaku (behavior). Tujuan OOP adalah membuat kode lebih terstruktur, reusable, dan mudah dikembangkan.

2. Konsep Kelas dan Objek

  • Kelas adalah blueprint atau cetakan.
  • Objek adalah hasil dari kelas.
  • Atribut adalah variabel dalam kelas.
  • Metode adalah fungsi dalam kelas.

3. Contoh Kelas Car

class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}

4. Constructor di Dart

Constructor digunakan untuk menginisialisasi atribut saat objek dibuat.

class Car {
  String? model;
  String? color;

  Car(this.model, this.color);
}

5. Praktik Kelas Mahasiswa

class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk);

  void tampilkanData() {
    print('Nama : $nama');
    print('NIM  : $nim');
    print('IPK  : $ipk');
  }
}

6. Encapsulation (Person)

class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  void setName(String name) {
    _name = name;
  }

  String getName() {
    return _name;
  }
}

7. Jawaban Quiz (Slide 14)

Perbedaan kelas dan objek:
Kelas adalah blueprint, objek adalah hasil dari blueprint tersebut.

3 Keuntungan OOP:

  • Kode lebih rapi dan terstruktur.
  • Dapat digunakan kembali.
  • Mudah dikembangkan.

Tugas Bonus - Kelas Hewan:

class Hewan {
  String nama;

  Hewan(this.nama);

  void bersuara() {
    print("$nama sedang bersuara");
  }
}

Kesimpulan

OOP membantu membuat program lebih terstruktur. Kelas adalah cetakan, objek adalah hasilnya. Constructor berfungsi untuk menginisialisasi data saat objek dibuat.
