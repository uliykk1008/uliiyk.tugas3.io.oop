Constructor

Constructor adalah method khusus yang otomatis dipanggil ketika objek dibuat untuk memberikan nilai awal pada atribut.

Default Constructor

Default constructor adalah constructor yang digunakan langsung saat objek dibuat tanpa perlu membuat constructor tambahan.

Named Constructor

Named constructor memungkinkan pembuatan objek dengan cara yang berbeda menggunakan nama constructor tambahan.

Constant Constructor

Constant constructor digunakan untuk membuat objek yang nilainya tidak berubah karena menggunakan atribut final.

Initializer List

Initializer list digunakan untuk menginisialisasi nilai atribut sebelum constructor dijalankan.

Public dan Private

Public dan private digunakan untuk mengatur akses terhadap atribut dalam class.

Getter dan Setter

Getter dan setter digunakan untuk mengambil dan mengubah nilai atribut secara terkontrol.

Enkapsulasi

Enkapsulasi adalah konsep OOP untuk menyembunyikan data dan memberikan akses melalui method.
