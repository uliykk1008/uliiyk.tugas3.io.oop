01. Constructor & Named Constructor

Constructor adalah method khusus dalam class yang otomatis dipanggil ketika objek dibuat. Constructor digunakan untuk memberikan nilai awal pada atribut objek.Selain constructor biasa, dalam Dart juga terdapat named constructor, yaitu constructor yang memiliki nama tambahan sehingga objek dapat dibuat dengan cara yang berbeda sesuai kebutuhan. .

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class Mobil {
  String? merk;
  int? tahun;

  Mobil(this.merk, this.tahun);

  Mobil.baru() {
    merk = 'Toyota';
    tahun = 2023;
  }

  Mobil.dariJson(Map json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  @override
  String toString() {
    return 'Mobil{merk: $merk, tahun: $tahun}';
  }
}

void main() {
  var mobilA = Mobil('Honda', 2020);
  print(mobilA);

  var mobilB = Mobil.baru();
  print(mobilB);

  var mobilC = Mobil.dariJson({"merk":"Suzuki","tahun":2021});
  print(mobilC);
}
}
}

02. Jenis-jenis Constructor di Dart

  • Default Constructor: Constructor bawaan yang otomatis dibuat oleh Dart jika tidak ada constructor lain di dalam class. Bentuknya ClassName().
  • Named Constructor: Constructor yang memiliki nama tambahan sehingga objek dapat dibuat dengan cara yang berbeda. Bentuknya ClassName.nama().
  • Parameterized Constructor: Constructor yang menggunakan parameter untuk memberikan nilai awal pada atribut ketika objek dibuat. Bentuknya ClassName(this.param).
  • Constant Constructor: Digunakan untuk membuat objek yang bersifat konstan dan biasanya digunakan jika atribut dalam class bersifat final. Bentuknya const ClassName().
  • Factory Constructor: Constructor yang digunakan untuk mengontrol proses pembuatan objek dan dapat mengembalikan objek baru atau objek yang sudah ada. Biasanya dibahas lebih lanjut pada materi lanjutan.

03. Constant Constructor

Constant constructor adalah constructor yang digunakan untuk membuat objek konstan. Constructor ini biasanya digunakan ketika semua atribut pada class bersifat final, sehingga nilai objek tidak dapat diubah setelah dibuat.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class Warna {
  final int r, g, b;
  const Warna(this.r, this.g, this.b);

  @override
  String toString() {
    return 'Warna(r: $r, g: $g, b: $b)';
  }
}

void main() {
  const merah = Warna(255, 0, 0);
  const merahLagi = Warna(255, 0, 0);

  print(merah);
  print(merahLagi);
}
}

04. Initializer List

Initializer list adalah cara untuk menginisialisasi nilai atribut sebelum constructor dijalankan. Teknik ini biasanya digunakan ketika nilai atributperlu diproses terlebih dahulu, misalnya mengambil nilai absolut dari suatu angka sebelum disimpan ke dalam atribut.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class Titik {
  final double x, y;

  Titik(double x, double y)
      : x = x.abs(),
        y = y.abs() {
    print('Titik dibuat di (${this.x}, ${this.y})');
  }
}

void main(){
  var titik1 = Titik(3.0, 4.0);
  var titik2 = Titik(-5.0, 10.0);
  var titik3 = Titik(-1.5, -2.5);

  print(titik1.x);
  print(titik2.x);
  print(titik3.x);
}
}

05. Public & Private

Public dan private digunakan untuk mengatur tingkat akses terhadap atribut atau method dalam sebuah class. Dalam Object Oriented Programming, atribut yang diawali dengan tanda _ dianggap private, artinya atribut tersebut hanya dapat diakses dari dalam class yang sama.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class Person {
  String _name;
  int age;

  Person(this._name, this.age);
}

void main() {
  Person person = Person("Salma", 20);
  print(person.age);
}
}

06. Getter

Getter adalah method yang digunakan untuk mengambil atau membaca nilai atribut dari sebuah class. Getter biasanya digunakan ketika atribut bersifat private sehingga nilai tersebut tidak dapat diakses secara langsung dari luar class.

07. Setter

Setter adalah method yang digunakan untuk mengubah atau memperbarui nilai atribut dalam sebuah class. Dengan setter, perubahan nilai atribut dapat dikontrol sehingga data tetap valid.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  set setName1(String name) {
    _name = name;
  }

  void setName(String name) {
    _name = name;
  }

  String getName() {
    return _name;

void main() {
  Person person = Person("Salma", 20);

08. Encapsulation pada Product

Encapsulation adalah konsep dalam OOP yang digunakan untuk menyembunyikan data dan mengatur akses terhadap data tersebut melalui method tertentu. Pada contoh produk, atribut disembunyikan dan hanya dapat diakses melalui getter dan setter yang memiliki validasi tertentu.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class Produk {
  String _nama;
  double _harga;

  Produk(this._nama, this._harga);

  String get nama => _nama;
  double get harga => _harga;

  set nama(String value) {
    if (value.length >= 3) {
      _nama = value;
    }
  }

  set harga(double value) {
    if (value > 0) {
      _harga = value;
    }
  }
}

void main() {
  var produk1 = Produk('Monitor Samsung', 3500.0);

  print(produk1.nama);
  print(produk1.harga);

  produk1.nama = 'Keyboard';
  produk1.harga = 1200.0;

  print(produk1.nama);
  print(produk1.harga);
}
}

09. Encaptulation pada Login System

Encapsulation juga dapat digunakan dalam sistem login untuk melindungi data penting seperti password. Pada contoh ini, password tidak dapat diakses langsung dari luar class. Pengguna hanya dapat melakukan proses login atau mengganti password melalui method yang telah disediakan..

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class User {
  String _username;
  String _password;

  User(this._username, this._password);

  String get username => _username;

  bool login(String username, String password) {
    return _username == username && _password == password;
  }

  void gantiPassword(String lama, String baru) {
    if (_password == lama && baru.length >= 8) {
      _password = baru;
      print('Password berhasil diubah');
    }
  }
}

void main() {
  final user1 = User('admin', 'password123');

  print(user1.login('admin', 'password123'));

  user1.gantiPassword('password123', 'passwordBaru123');
}
}
