01. Constructor

Constructor adalah method khusus dalam sebuah class yang otomatis dijalankan saat objek dibuat. Fungsinya untuk memberikan nilai awal pada atribut atau properti objek. Dalam konsep Object Oriented Programming, constructor membantu proses inisialisasi objek agar objek langsung memiliki data ketika dibuat. Pada bahasa pemrograman seperti Dart, constructor biasanya memiliki nama yang sama dengan class dan dapat menerima parameter untuk mengisi nilai atribut.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class Mahasiswa {
  String nama;

  Mahasiswa(this.nama);

  void info() {
    print("Mahasiswa: $nama");
  }
}

void main() {
  var m1 = Mahasiswa("Uliya");
  m1.info();
}
}

07. Encapsulation

Teknik keamanan data dengan menyembunyikan akses langsung ke variabel internal menggunakan prefix underscore.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  void setName(String name) => _name = name;
  String getName() => _name;
}

09. Product Class

Penggunaan Named Parameters dan Required untuk menciptakan objek yang lebih deskriptif dan aman.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class Produk {
  final String id, nama;
  final double harga;

  Produk({
    required this.id,
    required this.nama,
    required this.harga,
  });
}
