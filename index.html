1. Constructor

Constructor adalah metode khusus pada sebuah class yang otomatis dipanggil saat objek dibuat. Constructor digunakan untuk menginisialisasi nilai awal dari atribut dalam class.

2. Default Constructor

Default constructor adalah constructor yang langsung digunakan saat objek dibuat tanpa harus membuat constructor tambahan. Biasanya digunakan untuk memberikan nilai awal pada atribut class.

3. Named Constructor

Named constructor adalah constructor yang memiliki nama tambahan. Constructor ini digunakan untuk membuat beberapa cara pembuatan objek dengan kondisi yang berbeda.

4. Constant Constructor

Constant constructor digunakan untuk membuat objek yang nilainya tetap dan tidak dapat diubah. Constructor ini biasanya digunakan pada class yang memiliki atribut final.

5. Initializer List

Initializer list digunakan untuk menginisialisasi nilai variabel sebelum constructor dijalankan. Biasanya digunakan untuk atribut yang memiliki tipe final.

6. Public dan Private

Public dan private digunakan untuk mengatur akses terhadap atribut dalam class. Variabel dengan tanda underscore (_) bersifat private dan hanya dapat diakses di dalam class.

7. Getter dan Setter

Getter dan setter digunakan untuk mengambil dan mengubah nilai atribut secara terkontrol sehingga data tetap aman dan sesuai aturan yang dibuat dalam class.

8. Enkapsulasi

Enkapsulasi adalah konsep OOP yang digunakan untuk menyembunyikan data dan hanya memberikan akses melalui method tertentu agar data tetap aman.
